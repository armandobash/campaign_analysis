# campaign_analysis
This is a repository where I work on my campaign analysis for my portfolio.

Methodology
I structured the project in three stages:

1. Data cleaning and validation across the three provided datasets, SQL data consolidation, joining all sources into one unified table for exploration

2. Analysis to evaluate cross-channel performance.

3. Dashboard creation to evaluate cross-channel performance.

The SQL used for the merge is available in the project’s GitHub repository.
